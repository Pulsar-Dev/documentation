# PIXEL.Avatar ![](http://localhost:3000/client.png)

### Basic Example

  ```lua #
  function PANEL:Init()
    self.Avatar = vgui.Create("PIXEL.Avatar", self)
	self.Avatar:SetSize(PIXEL.Scale(50), PIXEL.Scale(50))
	self.Avatar:SetPos(PIXEL.Scale(10), PIXEL.Scale(10))
	self.Avatar:SetRounding(PIXEL.Scale(4))
	self.Avatar:SetDrawOnTop(true)
    self.Avatar:SetPlayer(LocalPlayer(), PIXEL.Scale(50))
  end
  ```

---

### Functions

##### `PANEL:SetRounding(number)`
  Used for setting the amount of rounding the corners of the avatar has.

  ###### Example:
  ```lua #
  function PANEL:Init()
    self.Avatar:SetRounding(PIXEL.Scale(10))
  end
  ```

##### `PANEL:SetPlayer(player, size)`
  Used for setting the avatars "player"

  ###### Example:
  ```lua #
  function PANEL:Init()
    self.Avatar:SetPlayer(LocalPlayer())
  end
  ```

##### `PANEL:PANEL:SetSteamID(player, size)`
  Used for setting the avatars "player" using a SteamID64

  ###### Example:
  ```lua #
  function PANEL:Init()
    self.Avatar:SetSteamID(LocalPlayer():SteamID64())
  end
  ```
