# PIXEL.Navbar ![](https://github.com/Pulsar-Dev/documentation/blob/main/public/client.png)

### Basic Example

  ```lua #
  function PANEL:Init()
    self.Navbar = vgui.Create("PIXEL.Navbar", self)
    self.Navbar:Dock(TOP)
    self.Navbar:SetTall(PIXEL.Scale(50))
    self.Navbar:AddItem("test1", "Test 1", function() notification.AddLegacy("Clicked 1!", NOTIFY_GENERIC, 5) end, 1, PIXEL.Colors.Gold)
    self.Navbar:AddItem("test2", "Test 2", function() notification.AddLegacy("Clicked 2!", NOTIFY_GENERIC, 5) end, 2, PIXEL.Colors.Diamond)
    self.Navbar:AddItem("test3", "Test 3", function() notification.AddLegacy("Clicked 3!", NOTIFY_GENERIC, 5) end, 3, PIXEL.Colors.Silver)
  end
  ```

---

### Functions

##### `PANEL:SetName(name)`
  Sets the name

##### `PANEL:SetColor(color)`
  Sets the color

##### `PANEL:GetItemSize()`
  Returns the item sizes

##### `PANEL:AddItem(id, name, doClick, order, color, icon)`
  Adds an item to the navbar
  ID Must be unique!

##### `PANEL:RemoveItem(id)`
  Removes an item from the navbar

##### `PANEL:SelectItem(id)`
  Selects an item
