# PIXEL.Category ![](https://raw.githubusercontent.com/Pulsar-Dev/documentation/main/public/client.png)

### Basic Example

  ```lua #
  function PANEL:Init()
    self.Category = vgui.Create("PIXEL.Category", self)
    self.Category:Dock(TOP)
    self.Category:DockMargin(PIXEL.Scale(10), PIXEL.Scale(10), PIXEL.Scale(10), PIXEL.Scale(10))
    self.Category:SetTitle("Categorys!")
  end
  ```

<Callout type="info" emoji="ℹ️">
  This panel is derived from [PIXEL.Button](http://localhost:5000/pixel-ui/vgui/button) \
  All functions available to [PIXEL.Button](http://localhost:5000/pixel-ui/vgui/button/) will work here.
</Callout>

### Functions

##### `PANEL:SetExpanded(bool)`
  Setting if the Category starts expanded or not

##### `PANEL:SetStartHeight()`
  Setting the category start height

##### `PANEL:SetAnimTime()`
  The time it takes for the category to open

##### `PANEL:SetPaintBackground(bool)`
  Setting if the category should paint the background or not

##### `PANEL:SetPadding()`
  The padding for the contents of the category

##### `PANEL:SetTitle(title)`
  Sets the title of the category

##### `PANEL:SetContents(contents)`
  Sets the contents of the category

##### `PANEL:OnToggle(expanded)`
  Ran when the category gets "toggled"

##### `PANEL:LayoutContent(w,h)`
  A replacement function for PANEL:PerformLayout() so we dont overwrite the default PerformLayout function
