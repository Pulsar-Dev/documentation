# PIXEL.Label ![](https://github.com/Pulsar-Dev/documentation/blob/main/public/client.png)

### Basic Example

  ```lua #
  function PANEL:Init()
	self.Label = vgui.Create("PIXEL.Label", self)
	self.Label:SetText("Hey " .. LocalPlayer():Nick() .. "!")
	self.Label:SetWide(PIXEL.Scale(500))
	self.Label:SetTall(PIXEL.Scale(100))
	self.Label:SetFont("PIXEL.Font.Size50")
  end
  ```

---

### Functions

##### `PANEL:SetText(string)`
  Sets the text on the label

##### `PANEL:SetFont(string)`
  Sets font for the text

##### `PANEL:SetTextAlign(number)`
  Sets the text alignment

##### `PANEL:SetTextColor(color)`
  Sets texts color

##### `PANEL:SetEllipses(bool)`
  Should the text have ellipses if its too long?

##### `PANEL:SetAutoHeight(bool)`
  Should the label set its height dynamically?

##### `PANEL:SetAutoWidth(bool)`
  Should the label set its width dynamically?

##### `PANEL:SetAutoWrap(bool)`
  Should the label wrap dynamically?
