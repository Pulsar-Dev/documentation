# PIXEL.ValidatedTextEntry ![](http://localhost:3000/client.png)

### Basic Example

  ```lua #
  function Init()
    self.ValidatedTextEntry = vgui.Create("PIXEL.ValidatedTextEntry", self)
    self.ValidatedTextEntry:Dock(TOP)
    function self.ValidatedTextEntry:IsTextValid(text)
        if text == "hi" then
            return true, "good"
        else
            return false, "bad"
        end
	end
  end
  ```
