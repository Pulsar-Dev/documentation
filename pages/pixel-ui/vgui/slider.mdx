# PIXEL.Slider ![](http://localhost:3000/client.png)

### Basic Example

  ```lua #
  function PANEL:Init()
    self.Slider = vgui.Create("PIXEL.Slider", self)
    self.Slider:Dock(TOP)
  end
  ```

<Callout type="info" emoji="ℹ️">
  This panel is derived from [PIXEL.Button](http://localhost:5000/pixel-ui/vgui/button/) \
  All functions available to [PIXEL.Button](http://localhost:5000/pixel-ui/vgui/button/) will work here.
</Callout>

---

##### `PANEL:OnValueChanged(fraction)`
  Runs when the value of the slider has been changed
