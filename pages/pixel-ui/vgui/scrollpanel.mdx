# PIXEL.ScrollPanel ![](https://github.com/Pulsar-Dev/documentation/blob/main/public/client.png)

### Basic Example

  ```lua #
  function PANEL:Init()
    self.ScrollPanel = vgui.Create("PIXEL.ScrollPanel", self)
    self.ScrollPanel:Dock(FILL)
    self.ScrollPanel:DockMargin(0, 0, 0, 0)
  end
  ```
