# Rounded Boxes

##### `PIXEL.DrawRoundedBox(borderSize, x, y, w, h, col)`
  Draws a rounded box

---
##### `PIXEL.DrawRoundedBoxEx(borderSize, x, y, w, h, col, topLeft, topRight, bottomLeft, bottomRight)`
  Draws a rounded box with corners that you can control

---
##### `PIXEL.DrawFullRoundedBox(borderSize, x, y, w, h, col)`
  Draws a full rounded box

---
##### `PIXEL.DrawFullRoundedBoxEx(borderSize, x, y, w, h, col, tl, tr, bl, br)`
  Draws a full rounded box with corners that you can control
