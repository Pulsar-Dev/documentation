# Fonts ![](https://github.com/Pulsar-Dev/documentation/blob/main/public/client.png)

##### `PIXEL.RegisterFontUnscaled(name, font, size, weight)`
  Registers a font that does not scale based on screen size

---
##### `PIXEL.RegisterFont(name, font, size, weight)`
  Registers a font that will scale based on screen size

---
##### `PIXEL.GenerateFont(size, weight, font, name)`
  Generates a font based on the given parameters, if no name is provided it will be "PIXEL.Font.Size(size)"

---
##### `PIXEL.GetTextSize(text, font)`
  Returns the size of text with the given font

---
##### `PIXEL.GetRealFont(font)`
  Gets the real font?
