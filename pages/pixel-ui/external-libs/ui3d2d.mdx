# [PIXEL.UI.UI3D2D](https://github.com/TomDotBat/ui3d2d) ![](https://github.com/Pulsar-Dev/documentation/blob/main/public/client.png)


##### `PIXEL.UI.UI3D2D.startDraw(pos, angles, scale, ignoredEntity)`
  Starts a UI3D2D rendering context in immediate mode
   - The ignoredEntity paramater is optional, this is used for disabling eyetrace collisions with the entity you're attaching your UI to.

---
##### `PIXEL.UI.UI3D2D.endDraw()`
  Ends a UI3D2D rendering context, only call this if you have called startDraw already and it has returned true

---
##### `PIXEL.UI.UI3D2D.isPressing()`
  Returns true if the user is holding down the UI input key.

---
##### `PIXEL.UI.UI3D2D.isPressed()`
  Returns true if the user started pressing the UI input key on this frame.

---
##### `PIXEL.UI.UI3D2D.getCursorPos()`
  Returns two numbers, the x and y values of the cursor position on the current UI.
  - Will return nil if the player isn't looking at the UI.

---
##### `PIXEL.UI.UI3D2D.isHovering(x, y, w, h)`
  Will return true if the cursor is currently within the bounds of the box specified in the parameters.

---
##### `PIXEL.UI.UI3D2D.drawVgui(panel :: Panel, pos :: Vector, angles :: Angle, scale :: number, ignoredEntity :: Entity)`
  Will draw a VGUI panel in 3D space and allow the user to interact with it.
  - The ignoredEntity paramater is optional, this is used for disabling eyetrace collisions with the entity you're attaching your UI to.
