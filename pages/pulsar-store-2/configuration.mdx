## MySQL
MySQL is not required to run Pulsar Store, but it is used for storing data when you have more than one server. If you only have one server, you can use SQLite is reccomended.

### MySQL Configuration
The MySQL configuration file can be found at the following location:
<FileTree>
	<FileTree.Folder name="pulsar-store" defaultOpen>
		<FileTree.Folder name="lua" defaultOpen>
			<FileTree.Folder name="pulsar-store" defaultOpen>

					<FileTree.File name="sv_mysql.lua" />

			</FileTree.Folder>
		</FileTree.Folder>
	</FileTree.Folder>

The configuration file is pretty self-explanatory, but if you need help, you can find it below:
```lua
PulsarStore.Config.MySQL.SQLEnabled = true -- Is MySQL enabled? If this is false, Pulsar Store will use SQLite instead.
PulsarStore.Config.MySQL.Host = "localhost" -- The host of the MySQL server.
PulsarStore.Config.MySQL.Username = "pulsar-user" -- The username of the MySQL server.
PulsarStore.Config.MySQL.Password = "Pa$$word1" -- The password of the MySQL server.
PulsarStore.Config.MySQL.Database = "pulsar-database" -- The database of the MySQL server.
PulsarStore.Config.MySQL.Port = 3306 -- The port of the MySQL server.
```

## Currencies
Currencies are the different types of currency that you can use in Pulsar Store. You can have as many currencies as you want, but you need at least one. The default currency is Pulsar Credits.

<Callout emoji="⚠️">
	Removing the default currency is **NOT** recommended, as it is used for a lot of things.<br/>
	The addon may not function correctly if you remove the default currency.<br/>
	**No support will be given if you remove the default currency.**
</Callout>

## Changing Language
You can change the language of Pulsar Store by changing the language in the configuration file. The default language is English.
