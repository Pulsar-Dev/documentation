# Getting Started
To get started, you'll need to install two things into your `garrysmod/addons` folder.
1. PulsarStore
2. PIXEL UI

The folder structure should look like this:
<FileTree>
	<FileTree.Folder name="garrysmod" defaultOpen>
		<FileTree.Folder name="addons" defaultOpen>
			<FileTree.Folder name="pixel-ui" defaultOpen>
				<FileTree.Folder name="..." defaultOpen>
				</FileTree.Folder>
				<FileTree.File name="README.md" />
			</FileTree.Folder>
			<FileTree.Folder name="pulsar-store" defaultOpen>
				<FileTree.Folder name="..." defaultOpen>
				</FileTree.Folder>
				<FileTree.File name="README.md" />
			</FileTree.Folder>
		</FileTree.Folder>
	</FileTree.Folder>
</FileTree>


<Callout emoji="💡">
	The addon folder name must not contain capital letters on **Linux** servers and will fail to load if it does.
</Callout>

# Installing
To install the addons, simply drag and drop the folders into your `garrysmod/addons` folder.

## PulsarStore
1. Download the latest version of PulsarStore from the [releases page](https://www.gmodstore.com/market/view/pulsar-a-simple-ingame-store/versions)
2. Extract the contents of the zip file into your `garrysmod/addons` folder.
3. Ensure the addon name does not contain capital letters.

## PIXEL UI
<Callout emoji="💡">
	Please note that PulsarStore **REQUIRES** the PulsarDevelopment fork of PIXEL UI as it contains special features for PulsarStore that are not in the main version.<br/>
	It is recommended that you use version 1.3.2 of PIXEL UI as it is the last known working version of PIXEL UI that works with Pulsar Store 1.
</Callout>

1. Download the version 1.3.2 of PIXEL UI from the [releases page](https://github.com/Pulsar-Dev/pixel-ui/releases/tag/1.3.2).
2. Extract the contents of the zip file into your `garrysmod/addons` folder.
3. Ensure the addon name does not contain capital letters.
