## Server Configuration
The server configuration file is located in `pulsar-store/lua/lyth_pulsar/sv_config.lua`.
This file contains all the server-side configuration options for the addon.

<Callout emoji="⚠️">
	The only thing that should be touched in this file is the `Lyth_Pulsar.Config.SV` table. Touching anything else may cause unintended issues such as data loss.
</Callout>

### Configuration Options

```lua
Lyth_Pulsar.Config.SV = {
    DefaultAmount = 0, -- This is a number which defines how many credits the users starts with on their first join.
    MySQL = {
        DBType = "sqlite", -- Set this to "mysql" if you want to use MySQL. If you do not know what this is i recommend you leave it as "sqlite".
        Hostname = "",  -- The hostname of the MySQL server. This is only used if you have DBType set to "mysql".
        Username = "",	-- The username of the MySQL server. This is only used if you have DBType set to "mysql".
        Password = "",	-- The password of the MySQL server. This is only used if you have DBType set to "mysql".
        Database = "",	-- The database of the MySQL server. This is only used if you have DBType set to "mysql".
        Port = 3306,	-- The port of the MySQL server. This is only used if you have DBType set to "mysql".
        TablePrefix = "lythpulsar_"	-- The table prefix to use inside the database.
    }
}
```

## Shared Configuration
The shared configuration file is located in `pulsar-store/lua/lyth_pulsar/sh_config.lua`.
This file contains all the shared configuration options for the addon.

### Configuration Options

```lua
-- Discount Configuration
Config.DiscountItems = { -- This is a table of items that will be discounted.
	["Item Name"] = 0.5 -- This is the discount multiplier. 0.5 = 50% discount, 0.25 = 25% discount, etc.
}

-- F4 Menu Configuration - Only supports PIXEL F4.
Config.PIXELF4Integration = true -- Should a Pulsar Store tab be added to PIXEL F4?
Config.F4Name = "Credit Store" -- The name to show on the f4 menu sidebar
Config.F4Icon = "YWSjW8T" -- The icon to show on the f4 menu sidebar

-- Credits Configuration
Config.GiveCreditsPeriodically = false -- Should we give players credits every x minutes?
--[[
Example:
Config.GiveCreditsPeriodically = 10 -- Every 10 minutes
Config.GiveCreditsPeriodically = 60 -- Every hour
Config.GiveCreditsPeriodically = 60 * 24 -- Every day
--]]

Config.EnableConversion = true -- Enable the Money Conversion?
Config.EnableBuyingCredits = true -- Enable Converting Money to Credits?
Config.EnableSellingCredits = true -- Enable Converting Credits to Money?

Config.CreditPrice = 100 -- How much does 1 credit cost?
Config.SellItemsBack = true -- Allows the player to sell their permanant items back to the store
Config.SellBackReturn = 0.50 -- The ammount of credits the player will get back for selling back their permanant items (0.50 = 50% of the original price)

-- Other
Config.EnableConfetti = true -- Should confetti be enabled when a player purchases something?
Config.PreiodicallyAmount = 100	-- The amount of credits to give players every x minutes.
Config.ChatCommand = "!store" -- Chat command to open the store
Config.UITitle = "Pulsar Store" -- Title of the store
Config.NPCModel = "models/Humans/Group01/Male_04.mdl" -- Model of the NPC
Config.NPCOverheadName = "Pulsar" -- The Name That Goes Above The NPC Head
Config.Language = "en" -- Language of the store
--[[
    Languages Avalable:
    "en" - English
    "de" - German
    "es" - Spanish
    "tr" - Turkish
    "ru" - Russian
    "fr" - French
--]]
Config.DisablePermaTag = false -- Disable the [Perma] tag on items
Config.RainbowIfPerma = true -- If the item is permanent, make the item name rainbow
Config.UIFont = "Rubik" -- The font to display in all UI's. WARNING: Changing this may cause text to be missplaced!

-- Ranks that are allowed edit the stores config (add items, categorys, etc)
Config.AdminRanks = {
    ["superadmin"] = true,
    ["admin"] = true,
    ["owner"] = true
}


-- Package config
Config.ArmorDefault = 0 -- The default armor to assign to a player when they return a package
Config.DefaultDamage = 100 -- The default damage to assign to a player when they return a package
Config.HealthDefault = 100 -- The default health to assign to a player when they return a package
Config.JumpDefault = 200 -- The default jump power to assign to a player when they return a package
Config.DefaultModel = "models/Humans/Group01/Female_01.mdl" -- The default model to assign to a player when they return a package
```
